## Getting Started

### Prerequisites

-   Npm
-   Nodejs

### Installation

1. Clone repo

2. Di chuyen vao repo vua tai ve

```
cd <repo>
```

3. Cai dat NPM packages

```
npm install
```

4. Chay api

```
npm run start
```

hoac

```
npm start
```

De test cac chuc nang cua API, cac ban su dung [Postman](https://www.postman.com/)
